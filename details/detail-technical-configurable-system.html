⚙️ Configurable System

Country Configuration

Flexible multi-country economic system setup

  • Base country selection: Configure primary economic data source (GER, FRA, etc.)
  • Virtual partners: Add EUR, ROW as virtual trading partners
  • Population modes: Control which countries get full agent populations
  • Scale ratios: Independent scaling for simulation vs. real-world economies

Open Economy Setup

Configurable international trade and market access

  • Trade modes: CLOSED_DOMESTIC, VIRTUAL_PARTNERS, FULL_INTERNATIONAL
  • Export strategies: DATA-driven or custom export ratio configurations
  • Import sources: VIRTUAL_MARKET or real supplier networks
  • FX systems: FIXED or dynamic exchange rate mechanisms

Firm Population Modes

Flexible agent granularity and sector coverage

  • SECTOR_ONLY: 20 sector-level units for broad economic analysis
  • HYBRID_MERGED: Detailed divisions for target sectors (2,3,4), sector-level for others
  • FULL_SECTOR_DIVISION: Complete 72-division granularity across all sectors
  • Reseed policies: SKIP, RECONCILE, or REPLACE existing firm populations
